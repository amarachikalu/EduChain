# EduChain - Educational Content Contribution System

A blockchain platform for contributing educational content, rewarding educators, and distributing knowledge tokens.

## Features
- Educational content submission
- Content coordinator oversight
- Knowledge token distribution
- Learning reward claiming

## Smart Contract Functions
- Submit educational materials
- Distribute knowledge tokens
- Claim learning rewards
- Track education statistics

## Usage
Promotes educational content creation through blockchain-verified contributions and knowledge-based rewards.
